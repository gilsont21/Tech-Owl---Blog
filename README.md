# Tech Owl Blog

Este é um projeto de blog simples desenvolvido para a matéria de Programação Web. O site permite visualizar artigos e postar comentários.

## Funcionalidades

* Visualização de artigos separados por categoria.
* Sistema de comentários funcional com PHP e MySQL.
* Prevenção de SQL Injection usando *Prepared Statements*.
* Design responsivo (básico) com HTML5 e CSS3.

## Tecnologias Utilizadas

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Banco de Dados:** MySQL

## Como Executar Localmente

1.  Você precisará de um servidor local (XAMPP ou WAMP).
2.  Coloque os arquivos do projeto na pasta `htdocs` (do XAMPP).
3.  Inicie o Apache e o MySQL.
4.  Crie um banco de dados no phpMyAdmin chamado `comentarios_blog`.
5.  Importe o arquivo `.sql` (você precisa criar um) para criar a tabela `comentarios`.
6.  Acesse `http://localhost/Tech-Owl-Blog` no seu navegador.
